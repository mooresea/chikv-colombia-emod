Geography
---------

This file holds the different available geographies and methods to set the parameters relative to the climate files appropriately.
The ``geographies`` dictionary associate a given geography with its parameters defining the different climate files and parameters along with specific demographics parameters.

Currently supported geographies include:

* Garki_Single
* Namawala
* Sinazongwe
* GwembeSinazongwePopCluster
* Dielmo
* Ndiop
* Thies
* Mocuba
* West_Kenya
* Solomon_Islands
* Solomon_Islands_2Node
* Nabang
* Tha_Song_Yang
* Malariatherapy
* Birth_Cohort
